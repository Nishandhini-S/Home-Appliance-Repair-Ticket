# HOME APPLIANCE REPAIR TICKET & TECHNICIAN VISIT SCHEDULING SYSTEM

The Home Appliance Repair Ticket & Technician Visit Scheduling System is a console-based Java application built using JDBC that allows a home appliance service company to:

   - Register and manage customers
   - Register and manage technicians
   - Create and track service tickets
   - Assign technicians and schedule visits
   - Update visit outcomes and close tickets
   - Enforce strict business rules and transaction safety

The system separates:
     -  Master Data → Customers & Technicians
     -  Transactional Data → Service Tickets
